### Heading Structure

Each weekly file follows this format:

- **Day Entry** — `## [Day] [Date]`
  - **News** — `### News`
    - Screenshots
    - Global News
    - Semianalysis
    - TLDR
    - TBPN
  - **Work Todos** — `### Work todos:`
    - DONE
  - **TODO** — `### TODO:`
  - **Regular Todos** — `### Regular todos:`
  - **Action Items and Notes** — `### Action Items and Notes`

## Usage

Create a new weekly file each week following the naming convention `week_[month]_[start_day].md`. Use the heading structure above to keep entries consistent.
